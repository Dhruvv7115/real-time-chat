# Real-Time Chat App

A full-stack real-time chat application built with WebSockets for low-latency bi-directional messaging.

## Tech Stack
- **Frontend:** React, TypeScript, CSS
- **Backend:** Node.js, TypeScript, WebSockets (ws library)

## Features
- Real-time messaging with no page refresh
- Multiple users can chat simultaneously
- Typing indicator — shows when another user is typing (debounced)
- Clean frontend/backend separation

## Running Locally

### Backend
cd chat-app-be
npm install
npm run dev

### Frontend
cd chat-app-fe
npm install
npm run dev
